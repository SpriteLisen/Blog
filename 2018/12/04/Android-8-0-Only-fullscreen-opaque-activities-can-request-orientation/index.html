Android 8.0 Only fullscreen opaque activities can request orientation

Posted on 2018-12-04 Edited on 2018-12-19 In Android , 系统适配

最近在做项目的时候出现了一个问题, 在 Android 8.0 的手机上无故报了 Caused by: java.lang.IllegalStateException: Only fullscreen opaque activities can request orientation 异常, 8.0 适配踩雷, 记录一下.

  1. 产生原因

  最近在项目里引用了 Slidr 框架做了侧滑返回, 根据该框架要求, 需要把所有支持侧滑返回的 Activity 的主题都设置成如下样式 :

  也就是要将侧滑的 Activity 设置成透明, 从此, 神坑开始了…

  当我很开心的运行项目, 立马就闪退了, 然后给我来了一个从来没见过的异常 :

Caused by: java.lang.IllegalStateException: Only fullscreen opaque activities can request orientation

  意思是 非全屏的不透明的 Activity 不允许设置屏幕方向.

  好了, 做了这么久安卓从来没听说过这一茬, 啥时候不是全屏的非透明 Activity 还不能设置方向了??
