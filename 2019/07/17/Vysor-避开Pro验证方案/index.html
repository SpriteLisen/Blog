Vysor-避开Pro验证方案

Posted on 2019-07-17 In 软件hack

Vysor 是一款功能强大的手机投屏镜像操作软件, 它是一款 chrome 插件, 可惜国外的好插件都是需要付费的, 不过我们伟大的程序猿咋能如此将就呢? 开始我们的 hack 之旅~

  1. 获取 Vysor 本地文件

  由于 Vysor 是一款 chrome 插件, 因此我们可以很轻易的在 chrome 插件商店安装它, 同时我们也可以很轻易的在本地找到他的文件.

  刚开始咱们也不知道它具体存在系统中哪个路径下, 但是我们可以通过 chrome 来查询 chrome 自身的 资料路径

  我们只需要在 chrome 的地址栏内输入 chrome://version 即可, 接下来会进入下面这个界面 :

图示如下:

  我们可以看到有一个叫做 个人资料路径 的东西, 在图中已经标红了.

  然后我们只需要复制该路径到资源管理器中按下伟大的回车键即可.

图示如下:

  可以看到存在一个 Extensions 文件夹, 我们双击进入. 进入后如下图 :

图示如下:

  然后我们就蒙圈了, 这些类似乱码的东西是啥?

  不要紧, 我们再在 chrome 的地址栏中输入 chrome://extensions , 接下来会进入到 chrome 的插件管理界面中, 如下图所示 :

图示如下:

  我们可以看到在最下方有一个 chrome 应用 子栏目, 这里我们可以看到 Vysor 的信息, 其中包含了一个 ID, 此时我们复制一下这个 ID, 然后回到资源管理器中, 使用我们强大的搜索功能, 找到其对应的文件夹, 结果如下 :

图示如下:

  1. 修改 Vysor 源码

  接下来我们进入该文件夹中, 然后进入其对应版本的文件夹中即可找到 Vysor 的源码文件了.

  进入之后, 我们再次使用搜索功能, 找到一个叫做 uglify.js 的文件, 并用一个代码编辑器将其打开. 打开之后我们会看到一堆混淆后的 js 代码.

  接下来我们去百度找一个在线格式化 js 的网站, 将源码帖进去, 然后重新格式化一下, 再将格式化好的代码复制到一个新的 js 文件中, 然后在该文件中搜索关键字 Vysor subscription is active. Thank you for your support, 找到后如下图所示 :

图示如下:

  找到后, 我们会发现最上面有几行代码 :

  接下来, 我们只需要在原来的 js 文件中把 g = !1 和 b = !1 修改为

g = 1 和 b = 1

  即可

  当我们再次打开 Vysor 的时候, 就已经不需要进行 Pro 验证了, 怎么样, 是不是很简单呢?
